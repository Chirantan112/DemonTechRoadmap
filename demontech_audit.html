Dashboard shows data for all 6 roadmaps simultaneously even if user hasn't started any
Only show roadmaps the user has actually started or bookmarked. The rest clutter the UI.
All achievement badges are permanently "Locked" for new users — no onboarding encouragement
Show the first badge as unlockable with a clear call-to-action ("Complete your first topic to unlock this").
Progress data is localStorage-only — cleared if browser data is cleared, no account, no sync
